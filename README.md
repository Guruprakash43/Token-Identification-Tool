# Token-Identification-Tool
A simple HTML tool built to speed up light-to-dark mode migration for the Nextwave learning portal. Instead of manually swapping color variables across legacy screens in Figma, developers can paste any hex code, filter by element type — text, background, border, icon, or action — and instantly see the matching light mode variable along with its dark mode counterpart.

Built using Claude Code by passing our design token JSON and the problem context. The color foundation is based on the Tailwind palette, so you can test the mapping logic with any Tailwind hex code.

How to use: Open the live link, enter a hex code, select the property type, and get your variable mapping.
